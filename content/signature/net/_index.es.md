El siguiente ejemplo de código muestra cómo aplicar una firma de texto a un documento PDF:

          using (Signature signature = new Signature("D:\\sample.pdf"))

              ForeColor = Color.Red
            };
            signature.Sign("D:\\signed.pdf", options);
          }
