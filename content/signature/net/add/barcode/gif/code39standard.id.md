steps:
    enable: true
    title_left: "Cara Menghasilkan CODE39STANDARD Barcode untuk file GIF di C#"
    content_left: |
        [GroupDocs.Signature](/id/signature/net/) memudahkan pengembang .NET untuk menambahkan kode batang CODE39STANDARD ke file GIF dalam aplikasi mereka dengan menerapkan beberapa langkah mudah.

        * Buat instance baru kelas Signature dan berikan jalur dokumen GIF sumber sebagai parameter konstruktor.
        * Buat instance objek BarcodeSignOptions dengan teks yang diperlukan dan setel properti EncodeType ke CODE39STANDARD.
        * Metode Tanda Panggil kelas Tanda Tangan dan berikan nama file GIF keluaran dengan BarcodeSignOptions ke dalamnya.
        * Analisis hasil SignResult untuk memeriksa tanda tangan yang baru dibuat jika diperlukan.
        
    title_right: "Persyaratan sistem"
    content_right: |
        GroupDocs.Signature untuk .NET API didukung di semua platform dan sistem operasi utama. Sebelum menjalankan kode di bawah ini, pastikan Anda telah menginstal prasyarat berikut di sistem Anda.

        * Sistem Operasi: Microsoft Windows, Linux, MacOS
        * Lingkungan Pengembangan: Visual Studio, Xamarin, MonoDevelop
        * Kerangka: .NET Framework, .NET Standard, .NET Core, Mono
        * Unduh versi terbaru GroupDocs.Signature untuk .NET dari [NuGet](https://www.nuget.org/packages/groupdocs.signature)
        
    code: |
        ```cs
        using (Signature signature = new Signature("sample.gif"))
        {
            // Inisialisasi opsi kode batang dengan teks kode batang yang telah ditentukan sebelumnya
            BarcodeSignOptions options = new BarcodeSignOptions("JohnSmith")
            {
                // atur jenis penyandian kode batang
                EncodeType = BarcodeTypes.Code39Standard,

                // atur posisi tanda tangan
                Left = 50,
                Top = 150,
                Width = 200,
                Height = 50
            };

            // Masuk file GIF dan simpan hasilnya 
            signature.Sign("signed.gif", options);
        }
        ```

          content: |
            Code 39 dikembangkan pada tahun 1974 oleh David Allais dan Raymond Stevens, kemudian dengan Interface Mechanisms Inc. (sekarang Intermec Corporation). Itu adalah simbologi barcode pertama yang menggunakan karakter alfabet selain angka numerik. Variasi Kode 39 telah digunakan secara luas di berbagai industri, terutama di militer AS sebagai komponen dari sistem Aplikasi Logistik Simbol Penandaan dan Pembacaan Otomatis (LOGMARS).
