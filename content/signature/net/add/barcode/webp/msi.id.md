steps:
    enable: true
    title_left: "Cara Menghasilkan MSI Barcode untuk file WEBP di C#"
    content_left: |
        [GroupDocs.Signature](/id/signature/net/) memudahkan pengembang .NET untuk menambahkan kode batang MSI ke file WEBP dalam aplikasi mereka dengan menerapkan beberapa langkah mudah.

        * Buat instance baru kelas Signature dan berikan jalur dokumen WEBP sumber sebagai parameter konstruktor.
        * Buat instance objek BarcodeSignOptions dengan teks yang diperlukan dan setel properti EncodeType ke MSI.
        * Metode Tanda Panggil kelas Tanda Tangan dan berikan nama file keluaran WEBP dengan BarcodeSignOptions ke dalamnya.
        * Analisis hasil SignResult untuk memeriksa tanda tangan yang baru dibuat jika diperlukan.
        
    title_right: "Persyaratan sistem"
    content_right: |
        GroupDocs.Signature untuk .NET API didukung di semua platform dan sistem operasi utama. Sebelum menjalankan kode di bawah ini, pastikan Anda telah menginstal prasyarat berikut di sistem Anda.

        * Sistem Operasi: Microsoft Windows, Linux, MacOS
        * Lingkungan Pengembangan: Visual Studio, Xamarin, MonoDevelop
        * Kerangka: .NET Framework, .NET Standard, .NET Core, Mono
        * Unduh versi terbaru GroupDocs.Signature untuk .NET dari [NuGet](https://www.nuget.org/packages/groupdocs.signature)
        
    code: |
        ```cs
        using (Signature signature = new Signature("sample.webp"))
        {
            // Inisialisasi opsi kode batang dengan teks kode batang yang telah ditentukan sebelumnya
            BarcodeSignOptions options = new BarcodeSignOptions("JohnSmith")
            {
                // atur jenis penyandian kode batang
                EncodeType = BarcodeTypes.MSI,

                // atur posisi tanda tangan
                Left = 50,
                Top = 150,
                Width = 200,
                Height = 50
            };

            // Masuk file WEBP dan simpan hasilnya 
            signature.Sign("signed.webp", options);
        }
        ```

          content: |
            MSI Plessey (juga dikenal sebagai Modified Plessey) adalah simbologi panjang variabel yang hanya berupa numerik yang merupakan varian dari Plessey Code, yang awalnya dikembangkan oleh Plessey Company of England pada tahun 1971. MSI Plessey biasanya digunakan untuk pelacakan inventaris, seperti di tepi depan rak di supermarket dan lingkungan gudang lainnya untuk menunjukkan di mana produk disimpan.
