        [GroupDocs.Signature for Java](https://products.groupdocs.com/signature/java/) facilita a los desarrolladores de Java la búsqueda de firmas Metadata en archivos Ods desde sus aplicaciones mediante la implementación de unos sencillos pasos.

        * Llame al método de búsqueda de la instancia de la clase Signature y pásele SearchOptions.

        // Set up input Ods file

        // Instantiate Signature for input file
